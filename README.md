<h1>Projeto E-commerce</h1>

<u><h2>LabeStore</h2></u>

Meu primeiro projeto utilizando o ReactJs, onde o objetivo é colocar em prática as técnicas aprendidadas de front-end de uma loja virtual, seguindo e garantindo requisitos propostos.

Neste projeto foram utilizados as seguintes ferramentas:

<li>JSX</li>
<li>Componentes</li>
<li>Props</li>
<li>Estado</li>
<li>Fluxo de dados no React</li>
<li>Renderização de Listas</li>
<li>Renderização condicional</li>

<h2> Função </h2>

O site é uma página de vendas de camisetas espaciais, onde o cliente consegue filtrar os produtos por nome e pelo preço, visualizar os produtos no carrinho, verificar o valor total da compra e informações sobre a loja, como formas de pagamentos, SAC, copyright etc.
